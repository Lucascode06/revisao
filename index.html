Revisão de HTML, CSS e JavaScript

HTML

HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas web. Aqui estão algumas tags e atributos importantes:

  • div: Usado para criar contêineres genéricos.

  • span: Usado para aplicar estilos em partes do texto.

  • id: Um identificador único para um elemento, usado em CSS e JavaScript.

  • class: Usado para aplicar estilos a um grupo de elementos.

  • P: Define um parágrafo

  • H1:Define um título 1

  • H2:Define um título 2

  • H3:Define um título 3

  • BR: Insere uma quebra de linha

  • HR:Define um filete horizontal

  • A:Define um hiperlink

  • Input:Cria campos de entrada em formulários

CSS

CSS (Cascading Style Sheets) é usado para estilizar páginas web. Sao algumas propriedades importantes:

  • display: Controla como um elemento é exibido (ex: block, inline, flex).

  • color: Define a cor do texto.

  • background-color: Define a cor de fundo de um elemento.

  • color:Define a cor do texto dentro de um objeto.Apenas do texto.

  • font-size:Define o tamanho da fonte de um objeto

  • font-style:Define se um texto deve estar, por exemplo, em itálico

  • border:Dá as definições da borda de um objeto.

  • border-radius:Serve para definir o quão arredondada é a borda de um objeto. É uma estilização importantíssima para o beleza do seu website

  • padding:É quase a mesma coisa que margin, mas invertido: define a distância que um objeto deve ter para com a borda de si mesmo.

  • margin:Define o espaço que deve ter entre um objeto e outro. Pode ser utilizado sozinho ou separadamente, como, por exemplo, margin-left, margin-right, margin-top e margin-bottom;

  • height:Anda junto de width e define a altura de um objeto;

  • width:Define a largura de um objeto. Muito utilizado em divs e em imagens, por exemplo

JavaScript

JavaScript é uma linguagem de programação que permite criar interatividade nas páginas web. Aqui estão alguns conceitos básicos:

  • let: Declara uma variável com escopo de bloco.

  • const: Declara uma variável constante que não pode ser reatribuída.

  • if: Estrutura condicional que executa um bloco de código se uma condição for verdadeira.

  • whois: Não é uma função padrão do JavaScript, mas pode se referir a serviços de consulta de domínio.

  • Comparação: Comparam valores, como `==`, `===`, `!=`, `!==`, `>`, `<`, `>=`, `<=`.

  • Arrays: Coleções ordenadas de valores, acessíveis por índice.

  • Tipos de Dados: Inclui `string`, `number`, `boolean`, `null`, `undefined`, `symbol` e `object`.

GitHub

GitHub é uma plataforma de hospedagem de código-fonte que utiliza o sistema de controle de versão Git. É amplamente usado para colaboração em projetos de software.

  • Repositórios: Contêm arquivos e histórico de versões.

  • Branches: Permitem o desenvolvimento paralelo.

  • Pull Requests: Usados para revisar e mesclar mudanças.

  • Clone: Criar uma cópia local de um repositório remoto para trabalhar no código localmente.
