Tags HTML

As tags HTML são os elementos fundamentais que compõem um documento HTML. Elas são usadas para estruturar o conteúdo da página.

Estrutura Básica de uma Tag

Uma tag HTML é composta por uma tag de abertura e uma tag de fechamento:

            <nome_da_tag>Conteúdo</nome_da_tag>

Exemplos de Tags Comuns

  • <html>: Define o início e o fim de um documento HTML.
  • <head>: Contém informações sobre o documento.
  • <body>: Contém o conteúdo visível da página.
  • <h1> a <h6>: Usadas para cabeçalhos.
  • <p>: Define um parágrafo.
  • <a>: Define um link.

Exemplo de Código

            <html>
                <head>
                    <title>Título da Página</title>
                </head>
                <body>
                    <h1>Bem-vindo!</h1>
                    <p>Este é um exemplo de um parágrafo.</p>
                    <a href="https://www.exemplo.com">Clique aqui</a> para mais informações.
                </body>
            </html>
